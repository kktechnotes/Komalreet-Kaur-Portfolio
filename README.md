## About This Portfolio

I'm a Technical Writer with 11 years of experience. Over the years I've worked on various end user documentation, including enterprise software guides, API documentation, integration guides, configuration guides, and developer-facing content across different industries and teams.

This portfolio is a reflection of the skills I've honed over the years and includes both professional work and personal projects. The professional samples show how I write in a real enterprise setting, and personal projects show my personality as an independent learner, which keeps my skills current.

I have selected these projects to show the variation in content and information structure I bring as per the audience and documentation purpose. That's something I pay attention to on every project.

I spend time understanding the product before I write anything. That usually means testing it myself, reading the code if I can, and asking engineers questions while balancing bi-weekly release cadence. The adaptability and technical mindset has helped me evolve.
